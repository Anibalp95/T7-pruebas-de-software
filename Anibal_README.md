Instrucciones de compilación
============================

- Versión de python utilizada: 3.8.2
- Librerias utilizadas:
    - boto3 == 1.13.17
    - botocore == 1.16.17

Para instalar boto3 y botocore utilizar el comando: 

$ pip install boto3

Para correr las pruebas utilizar el comando:

$ python run_anibal.py
